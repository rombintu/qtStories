# qtStories

## Server
```
git clone https://github.com/rombintu/qtStories.git
cd qtStories/server
cp config.py.bak config.py
python3 -m venv venv
. venv/bin/activate
pip install -r requirements.txt
python main.py
```
## Client
```
cp config.py.bak config.py
```
